## what I'm building

### 🌿 AfterMa — Postpartum Care App
Because the fourth trimester is real and the world has been mostly quiet about it.
AfterMa is a postpartum care platform that holds space for new mothers; physically, emotionally, practically.
Built not for metrics; built for the moment a new mother feels less alone.

### 🔗 Conduit — Open Banking Intelligence Platform
Financial intelligence shouldn't be a privilege.
Conduit bridges open banking data with human-readable insight; making the system legible to the people who need it most.
Not another fintech dashboard; a translator between complex data and real human decisions.
